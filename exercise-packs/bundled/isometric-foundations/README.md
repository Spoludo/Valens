# Isometric Foundations Exercise Pack

Home-friendly first bundled exercise pack. It avoids dead hangs and does not assume a pull-up bar.

Included exercises: hollow body hold, plank, bear crawl hold, wall sit, horse stance, wall push, reverse table hold, single-leg glute bridge hold, calf raise hold, pike push-up hold, and single-leg balance.

Planner rule: use `movementPatternId`, `jointStress`, `fatigueCost`, and `progression` metadata. Do not special-case exercise names.
