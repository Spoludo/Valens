# Valens

**Valens** is an open-source Android application for adaptive strength, mobility, balance and joint-aware training focused on healthy aging.

Valens is not a bodybuilding app, a calorie tracker, or a generic interval timer. It is a privacy-first, offline-first training coach designed to help people preserve physical capacity over decades.
