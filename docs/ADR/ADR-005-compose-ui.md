# ADR-005 – Jetpack Compose UI

**Status:** Proposed  
**Date:** 2026-07-05

## Context

Valens is a new Android project and should use modern UI architecture.

## Decision

Use Jetpack Compose and Material 3 for UI.

## Consequences

Positive:

- modern declarative UI
- strong Android future support
- easier state-driven screens

Negative:

- Compose testing and performance require discipline
